# Testing 目錄索引

本目錄存放測試流程與案例設計文件，負責把 Spec 與 API 契約轉成可執行的驗證計畫。

## 目錄內容

- `test-flow-template.md`：測試流程模板。
- `test-flow-<feature-name>.md`：單一功能測試流程文件。

## 建議命名

- 檔名格式：`test-flow-<feature-name>.md`
- `feature-name` 應與對應 Spec 維持一致。

## Agent 快速搜尋建議

- 先讀模板：`test-flow-template.md`
- 再讀同名功能文件：`test-flow-<feature-name>.md`
- 驗證時優先檢查是否已對應 Spec 驗收標準與 API 主要錯誤碼。
