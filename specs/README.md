# Specs 目錄索引

本目錄存放功能規格（Spec）文件，提供 PM 與下游角色（RD / QA）對齊需求的單一事實來源。

## 目錄內容

- `spec-template.md`：規格文件模板。
- `spec-<feature-name>.md`：單一功能的正式規格文件。

## 建議命名

- 檔名格式：`spec-<feature-name>.md`
- `feature-name` 建議使用 kebab-case，例如 `workflow-run-export`。

## Agent 快速搜尋建議

- 先讀模板：`spec-template.md`
- 找特定功能：`spec-<feature-name>.md`
- 若有多份規格，優先比對文件中的版本、狀態與最後更新時間。
