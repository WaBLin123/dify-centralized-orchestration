# Design 目錄索引

本目錄存放系統設計與技術決策文件，提供 RD 在實作前的架構與邊界共識。

## 目錄內容

- `design-template.md`：設計文件模板。
- `design-<feature-name>.md`：單一功能的設計文件。
- `tech-stack.md`：全專案技術選型基準。

## 建議命名

- 設計檔名格式：`design-<feature-name>.md`
- 技術選型固定檔名：`tech-stack.md`

## Agent 快速搜尋建議

- 先讀 `tech-stack.md` 取得全域技術棧基準。
- 再讀 `design-<feature-name>.md` 了解功能層級設計與取捨。
- 如需新建設計文件，先參考 `design-template.md`。
